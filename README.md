## Questão
As equações gerais das duas retas _r_ e _s_ sao respectivamente:

- _r_: x + 4y - 7 = 0
- _s_: 3x + y + l = 0

Crie uma aplicação que verifique se as retas sao concorrentes, e se forem concorrentes determine o ponto comum as retas.
